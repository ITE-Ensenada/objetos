SPACE FIGHTERS es un juego de 2 jugadores donde cada
jugador contrala una nave.

JUGADOR 1=
W=Mover nave hacia arriba
S=Mover nave hacia abajo
A=Mover nave hacia la izquierda
D=Mover nave hacia la derecha
E=Disparar la ultimate *En la parte de arriba izquierda de la pantalla se mostrara un 
                        cuadro azul, este te avisara que la ulti casi esta lista para usarse,
                        cuando el cuadro se vuelva rojo ya podras usar la ulti, pero esta solo 
                        esta disponible por un corto tiempo*
F=Disparo principal 

JUGADOR 2=
FLECHA ARRIBA=Mover nave hacia arriba
FLECHA ABAJO=Mover nave hacia abajo
FLECHA IZQUIERDA=Mover nave hacia la izquierda
FLECHA DERECHA=Mover nave hacia la derecha
N=Disparar la ultimate *En la parte de arriba derecha de la pantalla se mostrara un 
                        cuadro azul, este te avisara que la ulti casi esta lista para usarse,
                        cuando el cuadro se vuelva rojo ya podras usar la ulti, pero esta solo 
                        esta disponible por un corto tiempo*
M=Disparo principal

*Los asteroides se destruiran al ser impactados por una bala*

*La barra de color verde es la vida de tu nave, cuando la barra este totalmente roja significa que tu nave ha sido destruida*

*Para ejecutar el juego abrir el archivo space_fighters.py*
